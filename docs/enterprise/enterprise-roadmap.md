# Enterprise Roadmap

## Implemented Baseline
- Username/password admin login with RBAC.
- Structured activity logs in admin API.
- Bot tests + smoke checks.
- CI, security, and deploy GitHub workflows.
- Static deploy + rollback scripts.
- SLO, incident, DR, and release runbooks.
- Optional web telemetry endpoint wiring.

## Next-Phase Upgrades
1. IdP-backed SSO + MFA (OIDC/SAML).
2. Centralized logs/metrics dashboard (Grafana + Loki/ELK).
3. Secret manager integration (Vault or cloud equivalent).
4. Staging parity environment and monthly DR game-days.
5. Web Application Firewall + automated penetration testing.

# Security Policy

## Supported Branch
- `main`

## Reporting a Vulnerability
- Email security findings to project administrators.
- Include: reproduction steps, impact, affected component(s), and suggested fix.

## Response Targets
- Initial acknowledgement: within 24 hours.
- Triage decision: within 72 hours.
- Remediation timeline based on severity.

## Severity Guidelines
- Critical: auth bypass, data exfiltration, RCE.
- High: privilege escalation, stored XSS, sensitive data exposure.
- Medium: significant abuse path, CSRF on privileged action.
- Low: minor information disclosure or hardening issue.
